> 	给你一个正整数列表 L, 如 L=[2,8,3,50], 输出L内所有数字的乘积末尾0的个数,如样例L的结果为2.(提示:不要直接相乘,数字很多,可能溢出)

``` python
# 2016-10-02

s = [0, 0]

def num_2(x):
    if(x % 2 == 0):
        s[0] += 1
        num_2(x/2)

def num_5(x):
    if(x % 5 == 0):
        s[1] += 1
        num_5(x/5)       
    
for i in L:
    num_2(i)
    num_5(i)
    
print min(s[0],s[1])
```
